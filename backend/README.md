## Installation

```bash
$ npm install
```

## Migration

```bash
$ npm run build
$ npm run migrate:up
$ npm run migrate:down
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## API Docs

URL: [http://localhost:3000/api/docs](http://localhost:3000/api/docs)

### Migrations

Run migrations

- npm run migration:run

Create migrations

- npm run migration:create --name=your_migration_name
